Require

Each project starts with requirements which reflect customer expectations.

Descriptive requirements are the first step to successful product.

Qualitio allows you to:

  • Check requirements and their relations on each level.
  • Attach test cases to testable items.
  • Follow requirements implementation status based on test results.
  • Follow dependency of requirements.
  • Keep everything in one place.
  • Integrate with an existing REQ management system.

Report

Testing results are worthless without good reporting. Well prepared report gives a bird's eye view on current situation.

With Qualitio you can:

  • Create informative and aesthetic reports.
  • Each report can be tailored to your specific needs with the meta language engine.
  • Extensive API can be used to create even more customized reports.

Execute

Good quality assurance follows whole life cycle of a project on each level.

Qualitio allows you to:

  • Execute test cases to follow requirements.
  • Trace all the bugs which influence the test cases.
  • Attach verbose comments for each test run.
  • Prepare test sets to execute them during the whole project life cycle.

Store

Quality depends on testing and testing depends on test cases.

With Qualitio you can:

  • Create, edit new test cases.
  • Review existing test cases.
  • Link tests to corresponding requirements.
  • Keep test cases organized in a directory tree.
  • And still keep everything in one place.

Overview

Qualitio is an open source web application designed for creating, storing and executing manual test cases effectively.

  • Provides powerful reporting and bug handling.
  • Covers whole QA process from designing the test cases to generating custom reports.
  • Qualitio has an extensive API which can be used on any operating system.
  • Can be adapted quickly to the changes in the QA process. New features can be added whenever we need them. It is not limited to out-of-the-box solutions only.
  • Integration with the 3rd party bug tracking systems (like Bugzilla) is easy. Bugs can be easily linked to the test cases.
  • Qualitio is powered by Django and Python so it's very easy to maintain and quick to change according to your request.
  • Test cases can be easily localized.
  • User defined strings can be used in test cases for frequently used phrases. Updating test cases is easy!
  • Reports can be generated for multiple test runs at once with a single click.
  • Qualitio saves a lot of time on automating or simplifying frequent tasks (especially reporting and test cases execution).
  • It works with all the major web browsers an any operating system.
