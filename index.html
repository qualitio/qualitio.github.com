Requirements module

Each project starts with requirements which reflect customer needs.

Descriptive requirements divided into different level of abstraction is the first step for successful product delivery.

Qualitio gives you an option to:

  • Check requirements and their relations on each level.
  • Attach test cases to testable items.
  • Follow requirements implementation status based on test results.
  • Follow requirements dependency.
  • Keep everything in one place.
  • Interface to existing REQ management system.

Reporting module

Testing results are worthless without good reporting. Well prepared testing summary gives bird's eye view on product situation.

Qualitio gives you an opportunity to create fully informative and not data overloaded reports. Each report could be tailored to your needs thanks to meta language engine.

Extensive API can be used to create even more customized reports.

Execution module

Good quality assurance follows whole life cycle of a project on each level.

Qualitio gives you an option to:

  • Execute test cases to follow requirement(s) implementation and quality state.
  • Follow all bugs which influence testing activities.
  • Prepare verbose notes for each test run.
  • Execute same test sets for whole project life cycle.

Store module

Each project needs testing activities to assure solid quality.

Solid quality means good test cases. Good test cases covers all application on each level.

Qualitio gives you an option to:

  • Prepare, edit and review test cases.
  • Attach accepted tests to suitable requirements.
  • Create a structure based on directory tree to distinguish different testing activities.
  • And still keep everything in one place.

Overview

Qualitio is a software which may be used for preparing, storing and executing manual test cases. It focuses on test cases creation and execution. It provides powerful reports and great bug handling.

It's what you need. Simple tool which is easy and allows you to speed up your work without overhead.

  • Qualitio covers whole QA process from designing test cases to generating custom reports.
  • Qualitio has an extensive API which can be used on any operating system.
  • Qualitio can be adapted quickly to the changes in the QA process. New features can be added whenever we need them. We are not limited to out-of-the-box solutions only.
  • Qualitio is integrated with 3rd party bug tracking systems (currently Bugzilla). Bugs can be linked easily to failed test cases.
  • Qualitio is Django / Python powered. It's very easy to maintain or change according to customer requests and could be tailored to individual needs really fast.
  • Test cases in Qualitio can be easily localized.
  • User defined strings can be used in test cases for frequently used phrases. Updating test cases is easy!
  • Reports can be generated for multiple test runs at once with a single click.
  • Qualitio saves a lot of time on automating or simplifying frequent tasks (especially reporting and test case execution). This means more time for testing.

It works with all the major web browsers.

Qualitio allows you to cooperate with such powerful tools like Bugzilla.
